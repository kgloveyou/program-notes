# npm发包

注册npm账号后，需要邮件确认后才能发布。

If you are a new user, **please confirm your account first!**

https://stackoverflow.com/questions/61591685/npm-403-forbidden-put-http-registry-npmjs-org-package-name-forbidden



注意：如果npm源是淘宝的话，会影响你发布模块，这时候需要换回npm官网的源。

```sh
npm config set registry https://registry.npmjs.org
```
