> 什么是 FBX 文件格式？

FBX (.fbx) 文件是一种用于交换 3D 几何和动画数据的格式。 您可以使用不同的程序来打开、编辑和导出高保真 2D 和 3D 文件。 FBX 文件用于电影、游戏以及增强现实和虚拟现实 (AR/VR) 开发。

加拿大软件开发商 Kaydera 最初开发了专有的 FBX 文件格式，称为 FLM，用于存储来自动作捕捉设备的数据。 FBX 文件存储完整动画场景的所有数据。 这些数据包括骨骼、网格、照明、相机和几何体。 最终，FLM 被改编为支持 3D 数据的可视化，例如场景中的角色。 该程序后来更名为 FilmBox，文件扩展名为 FBX。 Softimage3D、LightWave3D 和 TuboCAD 等公司采用了这种文件类型。 2002 年，Kaydera 将其更名为 MotionBuilder，2004 年，另一家加拿大公司 Alias 收购了 MotionBuilder。 2006 年，Autodesk 获得了该程序和 FBX 文件格式的所有权。

> 如何使用 FBX 格式

开发人员使用 FBX 模型在应用程序和平台之间交换详细的几何数据和动画数据，包括纹理和颜色。大多数 AR 和 VR 平台都可以轻松导入 FBX 文件而无需转换。

导入 FBX 文件时，您的程序可能不会导入某些类型的数据，例如多边形或单个灯光。同样，[导出纹理可能会出现问题](https://www.marxentlabs.com/fbx-files/#section5)。 FBX 的默认格式是二进制，但您可以以较旧的 ASCII 格式保存文件。您可以在记事本中打开 ASCII 文件，这对于故障排除很有用。但是，最好不要编辑文本文件，因为这样很容易造成代码乱码。

AR 和 VR 通常使用两种类型的 3D 文件：OBJ 和 FBX。 OBJ 是一种 ASCII 文件类型，不支持动画或对象层次结构或复杂对象各部分之间的父/子关系。阅读“[关于使用 OBJ 文件你需要知道的一切](https://www.marxentlabs.com/obj-files/)”以了解有关该格式的更多信息。 FBX 支持场景层次，并保留了物体的比例和一些关于材质的基本数据。

对于 AR 和 VR 用途，您可以通过在将原始文件从源编辑程序导出之前减少原始文件中的对象数量来优化 3D 文件。此外，虽然更多的多边形往往会提高图像质量，但您可以通过减少任一场景中的多边形数量来提高性能。

> 使用 FBX 文件格式的优点

FBX 交换 3D 图形、动画和虚拟现实文件的跨平台能力使其对与多个团队和应用程序一起工作的自由艺术家特别有用。 但是，内部和自由艺术家都可以利用这些 FBX 优势：

- 为大多数程序提供 2D 和 3D 支持以及高效的管道
- 在应用程序与内部和外部团队之间轻松传输数据以实现无缝协作
- 提供一种快速的方法来保留程序之间的骨骼动画、相机、光照，甚至一些纹理和颜色
- 使用轻量级或在线 FBX 查看器应用程序快速共享工作并响应更新和修复的评论
- 作为 3DS、Maya、MotionBuilder 和 Mudbox 等 Autodesk 产品之间的主要交换格式
- 通过使用免费的 Autodesk FBX 转换实用程序或编辑程序中的本机功能，转换为 OBJ、DXF、3DS 和 DAE 格式以满足您的项目需求

> FBX 格式的限制

尽管被 3D 艺术家广泛使用，但 FBX 有一些限制：

- 导入可能很慢。因此，只导出您需要的资产并在目标应用程序中试验设置，以实现最干净、最详细的导入。
- 一些用户报告使用静态网格的最佳结果。在导出和导入时保留其他属性依赖于用户对源程序和目标程序中设置的高级知识。
- FBX 是一种专有格式，具有封闭式许可证。 SDK 描述了 FBX 读写头文件。 2013 年，Blender 组织分析了 FBX 二进制文件格式规范。
  只有在 Autodesk 书面同意的情况下，开发人员才能将 SDK 包含在新产品中以供销售或内部使用。
- 不同的 3D 程序使用不同的语言处理连接：
  Autodesk 有一个用于 C++ 的 FBX 绑定，以及一个用于 Python 的附加绑定，用于 Autodesk API。 Blender 为 FBX 提供 Python 导入和导出脚本及其 Python SDK
- 来自 Open Ended Group 的 Field 是一个实验性数字艺术合作组织，它提供了一个 Java 库来打开 FBX 文件并提取特定元素。

> 关于 FBX 的常见问题

还有更多关于 FBX 文件格式的问题？ 您将在下面找到有关纹理、FBX 记录器和 FBX 文件格式的常见问题的答案。

**FBX 文件是否包含纹理？**
一般来说，FBX 等 3D 文件会保留详细的几何数据、网格以及一些简单的动画、相机和照明信息。 但是，纹理和材质数据更难导出。 如何渲染纹理取决于目标系统的个别特征。

**如何导出带有纹理的 FBX？**
您可以将纹理保存在对象中或“烘焙”到其中，类似于预渲染文件。 纹理数据不会单独存储或导出到 XML 或其他数据文件中。 以这种方式附加纹理会在目标程序中创建完整的材质。

导出时，您还可以将纹理映射到对象。 在纹理映射中，您确定 3D 对象上的表面的 2D 坐标，称为 UV 坐标。 然后将颜色和纹理映射到每个表面。 目标程序将适当的纹理与正确的表面相匹配。

**什么是 FBX 录音机？**
游戏玩家使用 FBX 录像机录制游戏活动的高质量高清视频，以供查看和分析。 通过编辑功能，玩家可以保存和分享喜欢的游戏。

**什么是 FBX 二进制文件格式规范？**
Blender 基金会于 2013 年发布了 FBX 二进制文件格式规范。该规范试图为那些想要扩大可以打开 FBX 文件的应用程序数量的人破译 FBX 背后的专有二进制代码。 这个简短的文档侧重于对 FBX 进行编码而不是对其进行解码。 该规范审查了 ASCII 文件结构、二进制文件结构、节点记录格式和属性记录格式。
