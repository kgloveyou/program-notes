### Other options for generating html forms

- [jsonforms.io](http://jsonforms.io/) — Customizable JSON Schema-based forms with React, Angular and Vue support out of the box
- [react-json-editor](https://ismaelga.github.io/react-json-editor/) — is a very similar project aiming to achieve reactJs form generation from json. Cons — UI is not that great and feature set is much lesser than react-jsonschema-form
- XForms — this is a classic standard for declaring the form definition and genrating forms. I haven’t seen a ReactJS library that implements this well.
- [http://winterfell.andrewhathaway.net](http://winterfell.andrewhathaway.net/) — Generate complex, validated and extendable JSON-based forms in React. I haven’t tried this.
- [react-schema-form](https://github.com/networknt/react-schema-form) — A React component for building Web forms from JSON Schema
