3.0 版本里将有一个基于 Proxy 的观察者，它会提供全语言覆盖的响应式跟踪。相比于 2.x 版本里基于 Object.defineProperty 的观察者，新的实现更加强大：

- 可以检测属性的新增和删除

- 可以检测数组索引的变化和 length 的变化

- 支持 Map、Set、WeakMap 和 WeakSet
