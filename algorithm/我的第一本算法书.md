# 第5章 安全算法  

## 5.3 哈希函数  

​	哈希函数的算法中具有代表性的是 MD5 ①、 SHA-1 ②和 SHA-2 等。其中 SHA-2 是现在应用较为广泛的一个，而 MD5 和 SHA-1 存在安全隐患，不推荐使用。  

①　Message Digest Algorithm 5 的缩写。  

②　SHA 是 Secure Hash Algorithm 的缩写。

## 7.2 素性测试  

p = 素数

n < p

n<sup>p</sup> mod p = n

“费马小定理”。根据是否满足费马小定理来判断一个数是否为素数的方法就是“费马测试”。
