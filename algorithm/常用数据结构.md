## Stack

```
stack.peek()
```

# C++ 常用数据结构

## Vector

### push_back( )

函数将一个新的元素加到vector的最后面，位置为当前最后一个元素的下一个元素
push_back() 在Vector最后添加一个元素（参数为要插入的值）：

```c++
//在容器中添加10
 
int num = 10;
vector<int> vec;
vec.push_back(num);

```

### pop_back( )

删除Vector容器中的最后一个元素
