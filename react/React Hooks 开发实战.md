## 6.1 constate

​		constate 是一个基于 React Hooks 和 React Context 的轻量级状态管理库。它的主要功能是将自定义 Hooks 的执行结果传送到 Context 中，并利用 React Context 通信机制将结果提供给子组件消费使用，从而实现跨组件的状态共享。

```bash
yarn add constate
```
