# 有用的组件

## Fullscreen

为了实现视频的全屏选项，我们将使用 `screenfull` Node模块，用于跟踪视图何时处于全屏状态，和来自 `react-dom` 的 `findDOMNode` 来指定哪个文档对象模型 (DOM) 元素将使用 `screenfull`  显示为全屏。  

```bash
yarn add screenfull
```

Full-Stack React Projects Modern web development using React 16, Node, Express, and MongoDB (Second Edition).pdf，P540.
