# 积累

1、文字前添加蓝色方块。

```tsx
import React from 'react';
import trans from '@/utils/translate';

interface Props {
  title: string,
}
const Title: React.FC<Props> = (props) => {
  return (
    <div style={{ marginBottom: '20px', verticalAlign: 'center' }}>
      <span
        style={{
          display: 'inline-block',
          width: '2px',
          height: '12px',
          backgroundColor: 'blue',
          marginRight: '10px',
        }}
      ></span>
      {trans(props.title)}
    </div>
  )
}

export default Title;
```

2、全局关键字

CSS3 定义了规范中每个属性都接受的三个“全局”关键字：`inherit`、`initial` 和 `unset`。

- inherit  

关键字 `inherit` 使元素的属性值与其父元素的属性值相同。 换句话说，它强制继承发生，即使在它通常不会运行的情况下也是如此。 在许多情况下，你不需要指定继承，因为许多属性自然继承。 尽管如此，`inherit` 仍然非常有用。

- initial  

关键字 `initial` 将属性的值设置为定义的初始值，这在某种程度上意味着它“重置”了该值。 比如`fontweight`的默认值是`normal`。 因此，声明 `font-weight: initial` 与声明 `font-weight: normal` 是一样的。

在你认为并非所有值都明确定义了初始值之前，这可能看起来有点傻。 例如，`color` 的初始值是“取决于用户代理”。 那不是你应该输入的时髦关键字！ 这意味着`color` 的默认值取决于浏览器中的首选项设置等。 虽然几乎没有人将默认文本颜色设置从黑色更改，但有人可能会将其设置为深灰色甚至鲜红色。 通过声明 `color: initial;`，你告诉浏览器将元素的颜色设置为用户设置的默认颜色。

- unset  

关键字 `unset` 作为 `inherit` 和 `initial` 的通用替代品。 如果该属性是继承的，则 `unset` 与使用 `inherit` 具有相同的效果。 如果该属性未被继承，则 `unset` 与使用 `initial` 具有相同的效果。



这些全局值可用于所有属性，但有一个特殊属性只接受全局关键字：`all`。

参考《CSS The Definitive Guide, 4th Edition.2017.pdf》
