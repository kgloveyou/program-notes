# semantic-segmentation-editor

https://github.com/Hitachi-Automotive-And-Industry-Lab/semantic-segmentation-editor

A web based labling tool for creating AI training data sets (2D and 3D). The tool has been developed in the context of autonomous driving research. It supports images (.jpg or .png) and point clouds (.pcd). It is a [Meteor](http://www.meteor.com/) app developed with [React](http://reactjs.org/), [Paper.js](http://paperjs.org/) and [three.js](https://threejs.org/).
